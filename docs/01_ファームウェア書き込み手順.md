# UIFlow2 v2.4.7 ファームウェア書き込み手順（M5Stack CoreS3）

M5Stack CoreS3 に **UIFlow2 v2.4.7**（MicroPython ベース）を M5Burner で書き込む手順です。
この作業は **1台につき最初の1回だけ** 行います。書き込み後は `mpremote` で
MicroPython スクリプトを転送・実行できるようになります。

対象 OS：Windows 11 ／ 所要時間：約 5〜10 分（ダウンロード時間を除く）

---

## 0. 必要なもの

| 項目 | 内容 |
| --- | --- |
| 本体 | M5Stack CoreS3 |
| ケーブル | USB Type-C ケーブル（**データ通信対応**のもの。充電専用は不可） |
| PC | Windows 11 |
| ソフト | M5Burner（M5Stack 公式の書き込みツール） |
| アカウント | M5Stack アカウント（UIFlow2 利用に必要。無料登録） |

> **ケーブルの注意**：100円ショップ等の「充電専用」ケーブルだと PC が CoreS3 を
> 認識しません。データ通信対応のものを使ってください。

---

## 1. USB ドライバの確認（初回のみ）

CoreS3 を USB でつなぐと、多くの場合 Windows 11 が自動でドライバを当てます。
もし後の手順でポート（COMx）が出てこない場合は、以下のドライバを入れてください。

- CP210x（Silicon Labs）または CH9102（WCH）の USB-シリアルドライバ

M5Stack 公式のドライバ配布ページ：<https://docs.m5stack.com/en/download>

確認方法：Windows の「デバイス マネージャー」→「ポート (COM と LPT)」に
`Silicon Labs CP210x...` や `USB-Enhanced-SERIAL CH9102...` が表示されればOKです。

---

## 2. M5Burner のインストールと準備

1. M5Burner をダウンロードしてインストールします。
   公式ダウンロードページ：<https://docs.m5stack.com/en/uiflow/m5burner/intro>
2. M5Burner を起動し、右上の人型アイコンから **M5Stack アカウントでログイン**します。
   （アカウントが無ければ「Sign up」から無料登録）

> UIFlow2 を使うため、ここでログインする M5Stack アカウントは、後で
> UIFlow2 にログインするアカウントと **同じもの** にしてください。

---

## 3. CoreS3 を書き込みモードにする

1. USB Type-C ケーブルで CoreS3 と PC を接続します。
2. 画面が消えている状態で、本体側面の **RST（リセット）ボタンを約3秒間長押し**します。
3. インジケータが **赤 → 緑** に変われば、書き込み（ダウンロード）モードに入っています。

> CoreS3 は RST ボタンの長押しで自動的にダウンロードモードへ入る回路を備えています。
> 緑点灯を確認してから次へ進んでください。

---

## 4. UIFlow2 v2.4.7 を書き込む

1. M5Burner の左側で機種カテゴリ **CoreS3** を選びます。
2. ファームウェア一覧から **UIFlow2** の **v2.4.7** を探し、**Download**（未取得の場合）します。
   - バージョンのプルダウンがある場合は **2.4.7** を選択してください。
3. **Burn**（書き込み）ボタンを押します。
4. ダイアログで以下を設定します。
   - **COM ポート**：CoreS3 のポート（例 `COM3`）を選択
   - **Wi-Fi 設定**：SSID とパスワードを入力（後から M5Burner の **Configuration** で変更可）
     - mpremote で使うだけなら Wi-Fi は未設定でも構いませんが、UIFlow2 のネットワーク機能を
       使う予定があれば入れておくと便利です。
5. **Start / Burn** を押すと書き込みが始まります。プログレスが 100% になり
   「Success」「Burn Successfully」等が出れば完了です。
6. 書き込み後、CoreS3 は自動で再起動します。USB 接続は保ったままで構いません。

---

## 5. 書き込み確認

CoreS3 が再起動して UIFlow2 の起動画面（ロゴ／メニュー）が出れば成功です。

PC 側からの疎通確認は、`mpremote` で行うのが簡単です（インストールは
`docs/00_Windows環境構築_VSCode.md` を参照）。

```bat
python -m mpremote devs
```

`COM3` のようにポートが表示されれば、スクリプト転送・実行の準備が整っています。
続けて Level1 サンプルを動かす場合は `Level1_samples/README.md` を参照してください。

---

## 6. うまくいかないときは

| 症状 | 対処 |
| --- | --- |
| M5Burner にポート（COMx）が出ない | データ通信対応ケーブルか確認 → USB ドライバ（CP210x/CH9102）を導入 → 別の USB ポートを試す |
| 書き込みが始まらない／途中で失敗 | RST 3秒長押しで**緑点灯（書き込みモード）**にしてから Burn する |
| Burn 中にエラーで止まる | 他アプリ（UIFlow デスクトップ、シリアルモニタ、別の M5Burner）がポートを占有していないか確認して閉じる |
| 書き込み後に画面が真っ暗 | 一度 POWER ボタンで電源オフ／オン、または RST を1回押してリセット |
| `mpremote devs` で出てこない | 書き込みモード（緑）のままになっていないか確認。RST を1回押して通常起動させる |
